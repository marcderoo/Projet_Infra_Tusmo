# How to launch own md doc  :

Install Node.JS on your computer and paste the following commands into the shell:

- npm install -g jsdoc-to-markdown
- cd static
- jsdoc2md appUtils.js > appUtils.md
- jsdoc2md confetti.js > confetti.md
